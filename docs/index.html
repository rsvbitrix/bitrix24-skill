Возможности

Features

💰

CRM

CRM

Сделки, контакты, компании, лиды. Воронка продаж с суммами. Поиск зависших сделок.

Deals, contacts, companies, leads. Sales pipeline with amounts. Find stuck deals.

✅

Задачи

Tasks

Список задач, создание, делегирование. Автоматические предупреждения о просрочках.

Task lists, creation, delegation. Automatic overdue warnings.

📅

Календарь

Calendar

Расписание на день/неделю. Встречи + задачи в одном виде. Подготовка к встречам.

Day/week schedule. Meetings + tasks in one view. Meeting preparation.

👥

Команда

Team

Статус сотрудников, отделы, структура. Кто чем занят, у кого просрочки.

Employee status, departments, org chart. Who's doing what, who's overdue.

💬

Чаты

Chat

Сообщения в чат Битрикс24. Уведомления сотрудникам.

Messages in Bitrix24 chat. Notifications to employees.

📁

Файлы и проекты

Files & Projects

Диск, документы, рабочие группы, рабочее время, лента новостей.

Drive, documents, workgroups, time tracking, activity feed.
