Создайте вебхук в Битрикс24

Create a webhook in Bitrix24

Откройте ваш портал Битрикс24 → Приложения → Разработчикам → Другое → Входящий вебхук. Скопируйте URL вебхука — он выглядит так:

Open your Bitrix24 portal → Applications → Developer resources → Other → Inbound webhook. Copy the webhook URL — it looks like:

https://your-portal.bitrix24.ru/rest/1/abc123xyz/

Выберите эти разрешения (чтобы всё работало):

Select these permissions (for full functionality):

crm task calendar user department im sonet_group disk timeman log
3

Send the webhook to the bot

Просто отправьте URL вебхука в чат с ботом. Он сохранит его и начнёт работать. Всё!
